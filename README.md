# 💰 MONIQ - Жеке қаржыны басқару платформасы

**Миссия:** Арманыңды есепке, жинағыңды әдетке айналдыру.

---

## 🚀 Жоба туралы
**MONIQ** — пайдаланушыларға өздерінің қаржылық мақсаттарын қоюға, жинақтарын есептеуге және прогресті бақылауға көмектесетін Full-stack веб-қосымша. 5 апталық оқу курсының қорытынды жобасы ретінде жасалды.

## ✨ Негізгі функциялары
* **Мақсаттарды басқару (CRUD):** Мақсаттарды қосу, көру және өшіру.
* **Сүзгілеу (Filtering):** Категориялар бойынша мақсаттарды іріктеу.
* **Калькулятор:** Мақсатқа 180 күнде жету үшін күнделікті жинақ сомасын есептеу.
* **Іздеу:** Атауы бойынша жылдам іздеу жүйесі.

## 🛠 Технологиялық стек
* **Frontend:** HTML5, CSS3, JavaScript (DOM Manipulation)
* **Backend:** Node.js, Express.js
* **Database:** PostgreSQL
* **API:** RESTful API (Fetch-сұраныстар)
